NLP是对语言进行结构化处理，一般常用在如下场景：
1. 翻译
2. 提取关键词
3. 补充文本内容
4. 词语关系分析（词性、语义等）
5. 智能文本

主要在处理时学要解决的问题如下：
1. 分词
2. 词嵌入（词向量）
3. sequence2sequence
4. position matrix

目前主要用的模型
1. RNN
2. CNN（Facebook）
3. attention（Google）
4. encoder-decoder
